To start the backend:

1. Open terminal in the project root
2. Run: cd src
3. Run: java -cp "target/classes;target/dependency/*" com.tapri.TapriApplication

Alternative - Run with Maven:

1. Open terminal in the project root  
2. Run: mvn spring-boot:run -Dspring-boot.run.main-class=com.tapri.TapriApplication
